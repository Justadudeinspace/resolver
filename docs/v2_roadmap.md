# v2 Roadmap

## Phase 1: Core v2 delivery (current)
- V2 Personal (language + mode) behind `FEATURE_V2_PERSONAL`.
- V2 Group moderation (admin-only, paid) behind `FEATURE_V2_GROUPS`.
- Telegram Stars group subscriptions (monthly/yearly/lifetime).
- Admin-only moderation logs and control panel.

## Phase 2: Expanded group tooling
- Welcome message templates when `welcome_enabled=1`.
- Rules display and inline updates when `rules_enabled=1`.
- Additional moderation heuristics (links, spam).
- Optional join spike detection with admin notifications.

## Phase 3: Reporting & observability
- Exportable moderation logs.
- Admin digest summaries.
- Health dashboard for subscription status and usage.

## Phase 4: Personalization
- Per-user tone presets (beyond current retry modifiers).
- Language-specific fallback templates for non-LLM mode.
